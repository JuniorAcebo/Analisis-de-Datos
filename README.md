# 📊 Proyectos de Análisis de Datos

Este repositorio contiene dos proyectos de análisis de datos desarrollados
como parte de la materia **Análisis de Datos** en la carrera de Ingeniería
en Sistemas de la **Universidad Tecnológica Privada de Santa Cruz (UTEPSA)**.
Ambos proyectos fueron desarrollados utilizando Python en Jupyter Notebook.

---

## 🔥 Análisis de Focos de Calor – Santa Cruz, Bolivia 2024

Análisis de la ocurrencia de incendios y focos de calor registrados durante
el año 2024 en distintos municipios del departamento de Santa Cruz, Bolivia,
a partir de datos satelitales proporcionados por la NASA. El trabajo fue
realizado en diciembre del año 2025.

📁 [`/analisis-incendios`](./IncendiosForestales)

---

## 📦 Análisis de un Sistema de Pedidos Internacionales

Análisis e integración de un dataset de pedidos internacionales compuesto
por 13 tablas relacionadas, con el objetivo de obtener una visión clara del
ciclo completo de compra y venta, identificando patrones en pedidos, pagos,
envíos y comportamiento de clientes.

📁 [`/analisis-pedidos`](./Logistica)
